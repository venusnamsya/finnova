# Finnova Landing page

## Project Overview
Finnova is a fintech company landing page website designed to showcase innovative financial technology solutions for businesses and individuals.
The landing page presents information about the company, services offered, features, testimonials, statistics, and contact information.

## Features
- Responsive landing page design
- Navigation menu
- About us section
- Services section
- Features section
- Testimonials section
- Statistics section
- Contact section
- Company image intergration
- Custom CSS styling

## Technologies used
- HTML
- CSS
- Git
- Github pages

## Project purpose
This project was created as part of a wed development learning exercise to practice HTML structure, CSS styling, and github deployment.
